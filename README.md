# Book Shelf

A 3D interactive bookshelf application built with separate Carousel and Library views.

## Features

- **Carousel View**: A 3D infinite carousel of books with immersive background effects.
- **Library View**: A traditional bookshelf layout.
- **Interactive**: Click on books to view details.
- **Responsive**: Adapts to different screen sizes.

## Tech Stack

- **Vite**: Fast frontend build tool.
- **Three.js**: 3D graphics library for rendering the bookshelf and carousel.
- **Vanilla JavaScript**: Core logic without heavy framework overhead.

## Setup & Installation

1.  Clone the repository.
2.  Install dependencies:

    ```bash
    npm install
    ```

## Usage

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Data Enrichment

The project includes a script to enrich book data:

```bash
npm run enrich
```
